# Lync Bridge AI Sourcing Copilot

A multi-agent AI sourcing prototype for Lync Bridge.

This prototype converts a buyer requirement into a sourcing-ready report using specialized agents:

- Trend Scout Agent
- Product Concept Agent
- Supplier Match Agent
- Landed Cost Agent
- Compliance Guardian Agent
- Quality Sentinel Agent
- Order Orchestrator Agent
- Buyer Pitch Agent
- Final Report Builder

## Features

- Buyer requirement input
- Trend intelligence
- Product concept generation
- Supplier matching and scoring
- Landed cost estimation
- Compliance checklist
- QC risk analysis
- Order milestone timeline
- Buyer pitch generation
- Markdown and printable HTML report export

## Tech Stack

- Python
- Streamlit
- Pandas
- OpenAI optional
- JSON/CSV mock data

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
